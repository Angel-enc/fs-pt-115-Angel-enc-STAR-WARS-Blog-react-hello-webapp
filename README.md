🌌 Star Wars Blog & Reading List
Web interactiva construida con React.js que consume la API oficial de Star Wars (SWAPI).

✨ Características:
Visualización de personajes, planetas y vehículos.

Sistema de "Favoritos" con persistencia de datos.

Diseño responsive y navegación fluida con React Router.

🛠️ Stack Tecnológico:
FrontEnd: React, HTML5, CSS3 (Bootstrap).

Herramientas: Context API para la gestión del estado global.
